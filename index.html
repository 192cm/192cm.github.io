01 LLM · RAG

Financial RAG System

한국 DART 공시 PDF에서 재무 정보를 찾기 위한 하이브리드 RAG 파이프라인입니다. ColPali 기반 비전 검색, BM25+semantic 텍스트 검색, BGE-M3 reranker, Gemini Vision 보정 에이전트를 결합했습니다.

  • Exact Match 0.8, LLM Judge 4.2/5.0
  • 표 경계 누락을 sliding-window로 재검색
Repository
02 Forecasting

Retail Demand Forecasting

매장·상품 단위 시계열 데이터로 30일 수요를 예측하고 재고 리스크를 추정했습니다. TFT와 XGBoost를 나란히 구성해 예측 성능과 불확실성 구간을 비교했습니다.

  • TFT MAPE 10.15%, RMSE 9.14
  • XGBoost quantile interval과 safety stock 산출
Repository
03 MLOps

Telco Churn Monitoring

통신사 고객 이탈을 예측하고 데이터 드리프트가 감지되면 재학습하는 운영 관점의 머신러닝 파이프라인입니다.

  • XGBoost 기반 이탈 분류, Recall 0.75
  • Evidently AI로 drift report와 retraining 자동화
Repository
04 Generative AI

Customizing LoRA for Diffusion Models

Stable Diffusion v1.5에 LoRA adapter를 파인튜닝하며 데이터 출처, rank, alpha 변화가 결과 이미지에 미치는 영향을 비교했습니다.

  • web-crawled, real, generated dataset 비교
  • before-after grid와 ablation artifact 생성
Repository
05 Research Tool

arxiv-tracker

LLM 관련 arXiv 논문을 매일 수집하고 Gemini로 한국어 요약을 생성해 FastAPI 웹 UI에서 검색·즐겨찾기할 수 있게 만든 개인 논문 레이더입니다.

  • cs.CL, cs.LG, cs.AI 기반 키워드 필터링
